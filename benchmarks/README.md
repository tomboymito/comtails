# Профилирование расчётного астрофизического контура

Этот каталог содержит сценарии базового профилирования Python-реализации
моделирования пылевого хвоста кометы.

## Этап 1: базовый профиль производительности

```bash
python benchmarks/stage1_profile.py
```

## Усиленный сценарий (больше частиц и временных узлов)

```bash
python benchmarks/stage1_profile.py --ntimes 40 --nsizes 20 --nevent 300 --nx 256 --ny 256
```

## Выходные артефакты

- `benchmarks/output/stage1/summary.json` — параметры запуска, тайминги и список «горячих» функций;
- `benchmarks/output/stage1/dust_tail_worker.prof` — cProfile-дамп для анализа узких мест орбитально-фотометрического контура.
