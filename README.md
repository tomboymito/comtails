# COMTAILS — моделирование пылевого хвоста кометы

COMTAILS — это научный программный комплекс для фотометрического и динамического моделирования
пылевых хвостов комет в гелиоцентрической системе координат. Текущая кодовая база объединяет
исходный Python-контур и новый нативный вычислительный baseline на C++ для ускорения расчётов и
подготовки CUDA-переноса.

## Научная постановка задачи

Модель рассчитывает:

- эволюцию ансамбля пылевых частиц методом Монте-Карло;
- вклад солнечной гравитации и давления излучения в орбитальную динамику частиц;
- двумерную фотометрическую матрицу яркости кометного хвоста;
- карту оптической толщины (optical depth) в плоскости неба;
- интегральные наблюдательные параметры (например, Afrho и звёздная/кометная яркость).

## Ключевые возможности

- Монте-Карло генерация пылевого выброса и кинематики частиц.
- Расчёт гелиоцентрической орбитальной динамики (решение уравнения Кеплера).
- Поддержка сценариев выброса (изотропный, направленный к Солнцу, активные области ядра).
- Интеграция с JPL Horizons для эфемерид.
- Наложение звёздного поля и формирование FITS-продуктов.
- Параллельные вычисления и нативный CPU baseline для ускорения.

## Профилирование Python-контура (этап 1)

Базовый запуск:

```bash
python benchmarks/stage1_profile.py
```

Результаты (тайминги + cProfile) сохраняются в `benchmarks/output/stage1/`.

## Нативное вычислительное ядро (этап 2+)

Нативный CPU baseline находится в `comtails_core/`:

- `comtails_core/README.md`
- `docs/how_to_run_cpp_ru.md`

В текущей реализации есть:

- интерполяция астрофизического профиля пылепроизводства `dM/dt`;
- сепарабельная свёртка фотометрической матрицы яркости хвоста;
- аккумуляция вклада пылевых частиц в поток и оптическую толщину;
- детерминированный отбор скоростей выброса;
- Monte Carlo v1/v2 шаги (в v2 — явная физическая нормировка `/nevent`);
- parity-бенчмарки и regression-проверка контрольных метрик.
